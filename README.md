# 02-Space-Shooter
Space Shooter project for MSCH-C220, 9/27/2020. 

Score caps for their respective different levels - 100, 250, 400, 600

After 600 you win!

A and D move left and right, and spacebar shoots, ESC quits. 

## Implementation
Built using Godot 3.2.2
